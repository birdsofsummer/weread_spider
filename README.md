# weread_spider


## 步骤

1. 取cookie
2. 解析cookie
3. 构造headers


若要从微信聊天记录中获取cookie,手机需root

## 取cookie方法

### 1

1. 微信聊天界面打开 [这个页面][1]
2. adb连手机拿到微信的cookie文件

### 2

1.  电脑打开 [创作平台][2]
2.  扫码取出cookie手动粘贴
